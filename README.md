## Блок 1
### Аналитика музыкального стримингового сервиса mosMusic

Задачи первого блока представлены папке `./part1`.  
Каждая задача находится в файле `task{X}.sql`, где `{X}` - номер задачи по порядку.  
В папке `./images` находятся результаты запросов sql скриптов. Например, в фалле `images/part1-task1-res.png` находится результат 1 запроса из первого блока заданий.
